# Quick Start - התחלה מהירה

## 🚀 3 שלבים פשוטים

### 1. התקן Node.js
הורד והתקן מ: https://nodejs.org/ (גרסה LTS)

### 2. התקן חבילות
```bash
npm install
```

### 3. הרץ את השרתים

**טרמינל 1:**
```bash
python app.py
```

**טרמינל 2:**
```bash
npm run dev
```

### 4. פתח בדפדפן
http://localhost:3000

---

## 📝 הערות

- **כרגע**: כל הקבצים מוכנים, אבל צריך להריץ את השרתים
- **אחרי ההתקנה**: תראה את האפליקציה החדשה ב-React
- **השינויים**: כל הדפים החדשים (Dashboard, QuickUpdate) יעבדו
