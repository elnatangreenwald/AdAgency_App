# איך להריץ את האפליקציה - מדריך מהיר

## ⚠️ חשוב: כדי לראות את השינויים, צריך להריץ את השרתים!

כרגע יצרתי את כל הקבצים והקוד, אבל כדי לראות את האפליקציה עובדת, צריך:

## שלב 1: התקנת Node.js

1. הורד Node.js מ: https://nodejs.org/
2. התקן את הגרסה LTS (מומלץ)
3. **חשוב**: הפעל מחדש את הטרמינל/CMD אחרי ההתקנה

## שלב 2: התקנת Dependencies

פתח טרמינל בתיקיית הפרויקט והרץ:

```bash
cd "C:\Users\Asus\Desktop\AdAgency_App"
npm install
```

זה יקח כמה דקות - זה מוריד את כל החבילות הנדרשות (React, Vite, Tailwind, וכו').

## שלב 3: הרצת השרתים

צריך להריץ **2 שרתים במקביל**:

### טרמינל 1 - Flask Backend:
```bash
python app.py
```
השרת ירוץ על: http://localhost:5000

### טרמינל 2 - React Frontend:
```bash
npm run dev
```
השרת ירוץ על: http://localhost:3000

## שלב 4: פתיחת הדפדפן

פתח את הדפדפן וגש ל:
**http://localhost:3000**

זה יראה לך את האפליקציה החדשה ב-React!

## בעיות נפוצות

### "npm is not recognized"
- Node.js לא מותקן או לא ב-PATH
- הפעל מחדש את הטרמינל אחרי התקנת Node.js

### Port already in use
- שרת אחר כבר רץ על הפורט
- עצור את השרת הישן או שנה את הפורט ב-`vite.config.ts`

### CORS errors
- ודא ש-Flask רץ על פורט 5000
- ה-proxy ב-Vite אמור לטפל בזה אוטומטית
