# סינון משימות שהושלמו מלוח השנה

## תיאור הפיצ'ר

משימות שהסטטוס שלהן "הושלם" לא יוצגו יותר בלוח השנה בדשבורד.

**חריג:** משימות יומיות (`is_daily_task`) ימשיכו להופיע בלוח השנה גם לאחר השלמתן, מכיוון שהמערכת מעבירה אותן אוטומטית ליום העבודה הבא.

## שינויים בקוד

### קובץ: `app.py`

**פונקציה:** `get_tasks_for_calendar()` (שורות 1263-1267)

נוסף סינון לפני הוספת משימה לרשימת המשימות המוצגות בלוח השנה:

```python
# סינון משימות שהושלמו (למעט משימות יומיות)
task_status = task.get('status', 'לביצוע')
is_daily_task = task.get('is_daily_task', False)
if task_status == 'הושלם' and not is_daily_task:
    continue
```

## לוגיקה

1. בדיקת סטטוס המשימה
2. בדיקה האם המשימה מוגדרת כמשימה יומית
3. אם הסטטוס הוא "הושלם" **וגם** המשימה אינה יומית - המשימה לא תוצג בלוח השנה
4. משימות יומיות שהושלמו ימשיכו להופיע (כי הן מועברות אוטומטית ליום הבא)
