# איך לבדוק את השגיאה

## הבעיה
אתה רואה הודעה "שגיאה בהתחברות עם Google. אנא נסה שוב." בדפדפן.

## מה לעשות

### שלב 1: בדוק את הקונסול של השרת

1. פתח את **הטרמינל/קונסול** שבו השרת רץ (איפה שראית `python app.py`)
2. נסה להתחבר עם Google שוב
3. תראה בקונסול שורות שמתחילות ב:
   ```
   Error in Google callback: ...
   ```
   או
   ```
   Traceback (most recent call last):
   ...
   ```

### שלב 2: העתק את השגיאה

העתק את **כל השגיאה** מהקונסול (הכל מתחילת `Error` עד הסוף).

### שלב 3: שלח לי

שלח לי את השגיאה המלאה ואתקן אותה.

## מה יכול להיות השגיאה?

1. **בעיה עם ה-credentials** - אולי לא מקבל token מ-Google
2. **בעיה עם שמירת credentials** - אולי בעיה ב-users_db.json
3. **בעיה עם המידע מ-Google** - אולי לא מקבל email או שם
4. **בעיה עם redirect_uri** - אולי עדיין לא תואם

אבל בלי לראות את השגיאה המדויקת מהקונסול, אני לא יכול לדעת מה הבעיה.

**בדוק את הקונסול ושלח לי את השגיאה!**
